# MAC_changer

## Easiest and beginer friendly tool

To Change the MAC addres of the connection untill the connection fails. 

### Root required

### Usage
```
git clone https://github.com/EngineerRancho/MAC_changer
cd MAC_changer
sudo su
python MAC_changer.py
```
